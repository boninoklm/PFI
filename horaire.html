Conception Web

Description : Ce cours permet d’apprendre les bases de la conception de sites web modernes en utilisant des langages comme HTML, CSS et JavaScript.

Ce que j’ai apprécié : J’ai adoré la créativité permise par ce cours, notamment lors de la conception de projets personnalisés et les Exemples offert.

Ce que j’ai moins apprécié : Les examens à l'écris sur la totalité du code à retenir.

Intro Programmation

Description : Ce cours se concentre sur le langage C# et ses structures de données et les algorithmes.

Ce que j’ai apprécié : La résolution de problèmes complexes était simple et satisfesante.

Ce que j’ai moins apprécié : La vitesse du cours était trop lente, trop de temp passé à corriger les exercices.
